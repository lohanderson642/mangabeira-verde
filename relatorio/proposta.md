# Proposta do Projeto

O projeto "Mangabeira Verde" prevê:
- Canteiros elevados para horta (usando madeira reciclada ou pneus)
- Composteira comunitária simples (caixas de madeira ou tambores)
- Parcerias com Prefeitura, Emater-PB, escolas e USFs
- Oficinas semanais de cultivo agroecológico

Materiais iniciais estimados: R$ 1.200,00 (sementes, ferramentas, madeira).
